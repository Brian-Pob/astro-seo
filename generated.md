# Example of generaterd SEO tags

```html
<!DOCTYPE html>
<html lang="en">
  <head>
	<!-- Generated SEO tags -->
    <title>A Very Descriptive Title</title>
    <link rel="canonical" href="http://localhost:3000/" />
    <link rel="icon" href="/favicon.ico" />

    <meta property="og:title" content="A Very Descriptive Title" />
    <meta property="og:type" content="website" />
    <meta property="og:image" content="https://user-images.githubusercontent.com/5182256/131216951-8f74f425-f775-463d-a11b-0e01ad9fce8d.png"/>
    <meta property="og:url" content="http://localhost:3000/" />
    <meta name="description" content="A heavily optimized description full of well-researched keywords."/>

    <meta name="robots" content="index, follow" />

    <meta name="twitter:card" content="summary_large_image" />
    <meta content="Tinker Tailor Soldier Spy" name="twitter:title" />
    <meta name="twitter:creator" content="@jonasmerlins1" />
    <meta content="Agent" name="twitter:description" />
    <meta content="https://user-images.githubusercontent.com/5182256/131216951-8f74f425-f775-463d-a11b-0e01ad9fce8d.png" name="twitter:image"/>
	<!-- End Generated SEO tags -->

	<!-- Other <head> tags -->
  </head>
  <body>
	<!-- Body content... -->
  </body>
</html>
```
